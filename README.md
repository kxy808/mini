# Mini — Budget App

Mini is a full-stack budgeting application built as a structured learning project to develop real-world software engineering skills.

The goal of Mini is to simulate how professional software products are designed, built, and deployed — from product planning to backend architecture, frontend development, and production deployment.

Through this project, I am learning and applying concepts such as:

- Product Requirements & MVP design
- API architecture and backend development
- Database schema design (PostgreSQL)
- Authentication & security (JWT)
- Financial allocation and budgeting logic
- Frontend application architecture (React)
- Modern UI systems and data visualization
- Progressive Web App (PWA) offline capabilities
- SaaS deployment and monetization strategy

Mini follows a structured development roadmap that mirrors how real engineering teams build software products.

This project is part of my journey to becoming a self-taught software developer while building the technical foundation required for future systems like **8-Link**, a next-generation Health Information System.
